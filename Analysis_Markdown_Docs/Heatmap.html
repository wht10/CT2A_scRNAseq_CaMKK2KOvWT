Utilizing a Heatmap to validate single-cell cluster identities with user-selected Marker Genes

Jonathan Perera, William Tomaszeswki

2022-07-25

This vignette reproduces the creation of a heatmap utilized for the validation or identification of cluster identities in a Seurat Object. Our purpose for this heatmap was to crosscheck and validate the cluster identities that we had manually annotated. For this approach, the user must have a list of genes that they hope to use to validate the identify of a cluster of cells that may be expected to surface through dimensionality reduction and clustering. It may also be used for the initial identification of cell-types based on expected marker gene expression levels. The standard Seurat workflow was used to process the data.

Converting Lists to Vectors

chr_to_array_gene <- function(x) {
    b <- length(x[[1]])
    y <- vector(mode = "character", length = b)
    for (i in 1:b) {
        y[i] <- x[[1]][i]
    }
    return(y)
}

filter_genes <- function(x, gene_feature = gene_features) {
    index <- match(x, gene_feature)
    x_matched <- gene_features[index]
    x_matched <- x_matched[!is.na(x_matched)]
    return(x_matched)
}

IV. Heat Map

The heat-map itself is the vertical concatenation of individual heat maps for each of the literature-based gene signatures through the use of the ComplexHeatMap package (Gu, Z. Complex heatmaps reveal patterns and correlations in multidimensional genomic data. Bioinformatics 2016.). For example, the gene signature for cDC1 cells (Xcr1, Cadm1, Cd36, Itgae, Cd24a, Rab7b) will be on the y-axis and the cluster cell types from the Seurat Object will be on the x-axis. The average expression of each gene in the signature is plotted for each cell-type.

Average Expression and Preparation

V. Creation of Combined HeatMap

This section will discuss the creation of the individual heat maps in preparation for the vertical concatenation in order to create the final heat map.

Function for creating heat maps

color_choice = viridis(n = 256, option = "inferno")
# This is a universal choice on color to maintain consistency throughout the
# heatmaps. This is may be changed within the function according to the user's
# preference.

create_heatmap <- function(matrix, celltype_name, order_of_celltype, color = color_choice,
    title = "Validation of Manual Annotations") {
    heatmap = Heatmap(matrix, name = "Gene Expression", cluster_columns = TRUE, column_split = order_of_celltype,
        show_column_dend = FALSE, cluster_column_slices = FALSE, column_title_gp = gpar(fontsize = 18),
        column_gap = unit(0.5, "mm"), column_title = title, column_order = cluster_ids,
        show_row_dend = FALSE, col = color, row_title = paste(celltype_name, "signature"),
        cluster_rows = FALSE, column_names_gp = gpar(fontsize = 18), row_names_gp = gpar(fontsize = 14),
        row_title_gp = gpar(fontsize = 14), row_title_rot = 0, show_column_names = TRUE,
        use_raster = TRUE, raster_quality = 1)
    return(heatmap)
}
